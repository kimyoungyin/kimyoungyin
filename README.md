## About
더 나은 UX를 위해 “코드의 품질과 성능”을 꾸준히 의심하고 개선하며, 사용자의 경험이 흔들리지 않도록 구조를 정돈하는 데 집중하는 Frontend Developer.

- 요청 흐름/상태 모델을 재정의해 중복·예측 불가능한 동작을 줄입니다.
- 성능 지표(INP/LCP)와 접근성(WAI-ARIA)을 제품 품질의 기본값으로 봅니다.
- 테스트/자동화로 변경 비용을 낮추고 안정적인 릴리스를 돕습니다.

## Recent Highlights
- **Harvesters-LALA**: 인증/세션 동시성 레이스를 구조적으로 정리하고 INP 80ms대, WAI-ARIA 기반 접근성까지 개선했습니다.
- **Myblog**: Next.js Data Cache + TanStack Query 캐시 계층 재설계로 읽기 요청 중복과 캐시 미스를 줄였습니다.
- **승룡이네집 도서 검색 시스템**: ImageKit CDN + preload(priority) 전략으로 Cold LCP를 약 40% 개선했습니다.
